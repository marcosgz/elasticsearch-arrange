# Elasticsearch::Arrange

IMPORTANT! It's under development

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'elasticsearch-arrange'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install elasticsearch-arrange
